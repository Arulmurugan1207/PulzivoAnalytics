🎯 Use Cases & Custom Events

Track conversions, forms, and custom actions

📊 Custom Events

Track any custom interaction or conversion:

// Track button clicks
window.STKAnalytics.trackEvent('button_click', { button: 'signup' });

// Track form submissions
window.STKAnalytics.trackEvent('form_submit', { form: 'contact' });

// Track purchases
window.STKAnalytics.trackEvent('purchase', { 
  amount: 99, 
  plan: 'pro',
  currency: 'USD'
});

👥 User Identification

Track authenticated users and their journey:

// Identify a logged-in user
window.STKAnalytics.identifyUser('user_123', {
  email: 'user@example.com',
  plan: 'pro',
  signupDate: '2024-01-15'
});

📢 Campaign Tracking

Track promo impressions and clicks:

// Track promo impression
window.STKAnalytics.trackPromoImpression('promo_123');

// Track promo click
window.STKAnalytics.trackPromoClick('promo_123', 'learn_more');
